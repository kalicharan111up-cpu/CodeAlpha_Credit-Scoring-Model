# CodeAlpha_Credit-Scoring-Model
This repository demonstrates the model that makes predictions for credit worthiness of an individual. 


💳 Credit Scoring Prediction System

A complete end-to-end Credit Scoring Machine Learning System featuring:

✅ Multiclass credit score prediction (Poor, Standard, Good)
✅ Automated preprocessing + ML pipeline
✅ Model training with XGBoost / LightGBM / Random Forest
✅ Streamlit web application
✅ SHAP explainability for transparent AI decisions
✅ Modern UI with color-coded prediction cards
✅ Metadata-driven deployment pipeline

🎯 Objective

The goal is to predict a customer’s credit score category based on demographic and financial attributes.
The system uses a multiclass ML classifier and explains every prediction using SHAP for transparency.


🧠 Modeling Approach
Models Trained

Cleans data

Scales numerical variables
